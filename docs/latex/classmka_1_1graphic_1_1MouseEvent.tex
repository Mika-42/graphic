\doxysubsection{Detailed Description}
Mutable mouse state snapshot shared with {\ttfamily \doxylink{classmka_1_1graphic_1_1Window}{Window}} views. 

The state is updated by GLFW callbacks in {\ttfamily \doxylink{classmka_1_1graphic_1_1Window}{Window}} and read from user-\/facing {\ttfamily Window::\+\+Mouse\+Event\+View} in the render loop.
