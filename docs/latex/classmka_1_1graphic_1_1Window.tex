\doxysubsection{Detailed Description}
Base window abstraction hosting the render loop. 

Derived classes only implement {\ttfamily render()} while this class manages:\+
\begin{DoxyItemize}
\item GLFW init/\+teardown
\item API-\/specific window hints
\item \doxylink{classmka_1_1graphic_1_1Context}{Context} setup and buffer presentation
\item Projection updates on resize 
\end{DoxyItemize}
